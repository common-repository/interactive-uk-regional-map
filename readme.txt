== Installation ==

1. Install the plugin .zip file through your WordPress 'Plugins' screen directly, or Upload the plugin files to the `/wp-content/plugins/ukr-map` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Now, you will see a new menu button in the left side bar of your WordPress called 'UK Reg. Map', click it and customize your map then save the changes.
4. Use this shortcode `[ukr_map]` to display the map in any page or post.

== Frequently Asked Questions ==

= Can I add an image in the hover popup? =

Yes, you can write any text and add images.

= Does this map requires Flash to work? =

No, it's SVG based and works perfectly on desktop along with mobile devices 'smartphone'.

= Is this map responsive? =

Yes.

= Can I split England regions into smaller regions? =

No, but fortunately we already have an [Premium Interactive UK Map](https://www.wpmapplugins.com/interactive-uk-map-wordpress-plugin.html) with the 12 clickable regions of England plus Ireland.

= Can I set a region such as Ireland as inactive? =

Yes, simply un-check the (active) box beside that region to make it inactive.

= Can I use the shortcode in different pages to display the same map? =

Yes.

= Can I create a different map and use it in another page? =

No, but you can buy the **[Multiple Instances Regional UK Map Package](http://wpukr.html5maps.pay.clickbank.net/)** to get 3 instances of the plugin and install each one to get a different styled map.

= Can I link a region to any webpage? = 

Yes, also you can choose to open it in the same window or in a new window.
